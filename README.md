## Prompt Engineering Tips

### The CORE Framework
```
C - Context:    Set the scene and role for the AI
O - Objective:  State exactly what you want
R - Rules:      Define constraints and formatting
E - Examples:   Show what good output looks like
```

### Universal Modifiers
Add these to any prompt for better results:
- `"Think step by step"` — Forces chain-of-thought reasoning
- `"Be specific and detailed"` — Prevents vague responses
- `"Format as a table/list/code"` — Controls output structure
- `"Act as a [role]"` — Sets expertise context
- `"Give me 5 variations"` — Gets diverse options
- `"Critique your own answer"` — Triggers self-reflection
