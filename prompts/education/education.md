# Education Prompts

> Prompts for teachers, students, and lifelong learners — lesson planning, study aids, tutoring, and curriculum design.

**Best with:** ChatGPT, Claude, any LLM

---

## For Teachers

### 1. Lesson Plan Generator
```
Create a complete lesson plan for teaching [topic] to [grade level/audience].

Duration: [X minutes/hours]
Learning objectives: [what students should be able to do after]
Prerequisites: [what students should already know]

Include:
1. Warm-up activity (5 min) — Hook their attention
2. Direct instruction (15 min) — Core concepts with examples
3. Guided practice (15 min) — Worked examples together
4. Independent practice (15 min) — Students try on their own
5. Assessment — How to check understanding
6. Differentiation — Adjustments for advanced and struggling students
7. Materials needed
```

### 2. Quiz Generator
```
Create a quiz on [topic] for [level] students.

Include:
- 5 multiple choice questions (4 options each, one clearly best)
- 3 short answer questions
- 2 application/analysis questions
- 1 bonus/challenge question

For each question:
- Correct answer
- Explanation of why it's correct
- Common misconceptions that wrong answers represent

Difficulty distribution: 30% easy, 50% medium, 20% hard.
```

### 3. Rubric Builder
```
Create a detailed grading rubric for [assignment type: essay/project/presentation].

Topic: [assignment topic]
Total points: [X]

Create a table with:
- 4-5 criteria (rows)
- 4 performance levels: Excellent / Proficient / Developing / Beginning
- Specific descriptors for each cell (not vague)
- Point values for each level

Make descriptors objective enough that two graders would assign similar scores.
```

### 4. Differentiated Assignment
```
Design a differentiated assignment on [topic] for a class with mixed abilities.

Create three tiers:
1. **Foundation** — For students who need support (scaffolded, structured)
2. **Standard** — For on-level students (the core assignment)
3. **Extension** — For advanced students (deeper thinking, open-ended)

All three tiers should:
- Cover the same learning objective
- Be equally valued (not "more work" for advanced)
- Allow students to demonstrate understanding
- Include clear success criteria
```

## For Students

### 5. Concept Explainer
```
Explain [concept] as if you're the world's best tutor.

1. Start with WHY this matters (motivation)
2. Use an everyday analogy to build intuition
3. Give the formal definition with key terms bolded
4. Walk through 2 examples step by step
5. Show 1 common mistake and how to avoid it
6. Give me 3 practice problems (easy, medium, hard)
7. Connect this to what I might learn next

Keep it conversational, not textbook-dry.
```

### 6. Essay Outliner
```
Help me outline an essay on: "[essay topic/question]"

Thesis statement options (give me 3 to choose from):
1. [option 1]
2. [option 2]
3. [option 3]

For the strongest thesis, create:
- Introduction (hook → context → thesis)
- Body paragraph 1 (topic sentence, evidence, analysis)
- Body paragraph 2 (topic sentence, evidence, analysis)
- Body paragraph 3 (counterargument and rebuttal)
- Conclusion (restate thesis, broader significance, final thought)

For each body paragraph, suggest specific evidence or examples I could use.
```

### 7. Study Guide Maker
```
Create a comprehensive study guide for [subject/chapter/exam]:

1. **Key Concepts** — Top 10 things I must know (1-2 sentences each)
2. **Vocabulary** — Essential terms with definitions
3. **Formulas/Rules** — All formulas with when to use each
4. **Common Exam Questions** — 5 questions with model answers
5. **Memory Tricks** — Mnemonics, acronyms, or visual aids
6. **Quick-Review Checklist** — Can I explain each of these? (yes/no list)
7. **Practice Problems** — 5 problems I should try before the exam

Make it fit on 2 pages if printed.
```

### 8. Research Paper Assistant
```
Help me plan a research paper on [topic]:

1. **Narrow my topic** — Suggest 3 specific research questions
2. **Thesis development** — Help me form an arguable thesis
3. **Source strategy** — What types of sources do I need? How many?
4. **Outline** — Suggested structure with section descriptions
5. **Counterarguments** — What would critics say? How do I address them?
6. **Citation tips** — [APA/MLA/Chicago] formatting reminders
7. **Writing checklist** — What makes an A paper vs a B paper

Academic level: [high school / undergrad / graduate]
```

### 9. Math Problem Solver
```
Solve this math problem step by step. For each step:
1. State what you're doing in plain English
2. Show the mathematical operation
3. Explain WHY you did that step (not just what)

After solving:
- Check the answer by plugging it back in
- Show a shortcut method if one exists
- Give me a similar practice problem to try

Problem: [your math problem]
```

### 10. Flashcard Generator
```
Create flashcards for studying [topic].

Generate 20 flashcards in this format:
**Front:** [question or term]
**Back:** [answer or definition + one memorable detail]

Mix question types:
- Definition cards (what is X?)
- Application cards (when would you use X?)
- Comparison cards (how is X different from Y?)
- Example cards (give an example of X)

Order from foundational concepts to advanced. Make backs concise but complete.
```

## Curriculum & Course Design

### 11. Course Syllabus
```
Design a [X-week] course syllabus for [course title].

Audience: [who — undergrad, professional, self-learner]
Prerequisites: [what they should know]
Learning outcomes: [what they'll be able to do]

For each week provide:
- Topic and subtopics
- Learning objectives (measurable, using Bloom's taxonomy verbs)
- Required reading/materials
- Activities/assignments
- Assessment method

Include: grading breakdown, course policies, recommended textbooks.
```

### 12. Active Learning Activities
```
Design 5 active learning activities for teaching [topic] to [audience].

For each activity:
- Name and brief description
- Time needed
- Materials/setup required
- Step-by-step instructions
- How students demonstrate learning
- Debrief questions

Activity types to include:
- Think-Pair-Share variation
- Problem-based learning scenario
- Jigsaw or collaborative task
- Simulation or role play
- Technology-enhanced activity
```

### 13. Learning Path Designer
```
Create a self-study learning path for [skill/subject].

Starting level: [complete beginner / some experience / intermediate]
Goal: [what they want to achieve]
Time available: [hours per week]
Preferred format: [video / reading / hands-on / mixed]

Provide:
- Phase 1 (Foundation): Topics, resources, milestones
- Phase 2 (Building): Topics, resources, projects
- Phase 3 (Mastery): Topics, resources, portfolio pieces
- Estimated total time
- How to know when you're ready for the next phase
```
