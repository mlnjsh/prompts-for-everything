# ChatGPT / LLM Prompts

> Power-user prompts for ChatGPT, GPT-4o, and other LLMs. Productivity, analysis, creativity, and more.

**Best with:** ChatGPT (GPT-4o), GPT-4.5, any capable LLM

---

## Thinking & Analysis

### 1. First Principles Thinker
```
You are a first-principles thinker. I will describe a problem, and you will:
1. Break it down to its most fundamental truths
2. Question every assumption
3. Rebuild the solution from the ground up
4. Explain why conventional wisdom might be wrong

Problem: [describe your problem]
```

### 2. Devil's Advocate
```
I'm going to share my plan/idea. Your job is to argue AGAINST it as strongly
as possible. Find every weakness, flaw, risk, and reason it could fail.
Be brutally honest but constructive. After your critique, suggest how to
address each weakness.

My plan: [describe your plan]
```

### 3. Decision Matrix
```
Help me make a decision between [Option A], [Option B], and [Option C].
Create a weighted decision matrix with these criteria:
- Cost (weight: [X])
- Time to implement (weight: [X])
- Risk level (weight: [X])
- Long-term value (weight: [X])
- [Add your criteria]

Score each option 1-10 on each criterion, calculate weighted totals,
and give me a clear recommendation with reasoning.
```

### 4. Socratic Questioning
```
I want to deeply understand [topic]. Instead of explaining it to me directly,
ask me a series of Socratic questions that lead me to discover the answer
myself. Start simple and progressively go deeper. If I get stuck, give a
small hint but keep asking questions. Guide me to genuine understanding.
```

### 5. Steel Man Argument
```
I believe [your position]. Instead of attacking this view, give me the
absolute STRONGEST possible argument FOR the opposing view. Steelman it —
make it as compelling and persuasive as possible, with evidence and logic.
Then tell me what I should consider from this opposing view.
```

## Productivity & Work

### 6. Meeting Summary
```
Summarize the following meeting transcript/notes into:
1. **Key Decisions** (bulleted list)
2. **Action Items** (who | what | deadline)
3. **Open Questions** (unresolved issues)
4. **Next Steps** (what happens next)

Keep it concise. Maximum 1 page. Flag any conflicts or ambiguities.

Meeting notes: [paste notes]
```

### 7. Email Rewriter
```
Rewrite this email to be [more professional / more concise / more friendly /
more assertive]. Keep the core message but adjust:
- Tone: [specify tone]
- Length: [shorter / same / more detailed]
- Goal: [what reaction do you want from the reader]

Original email: [paste email]
```

### 8. Project Planning
```
I need to [describe project]. Act as a senior project manager and create:
1. Work Breakdown Structure (WBS) with all tasks
2. Dependencies between tasks
3. Estimated time for each task
4. Critical path identification
5. Risk register (top 5 risks with mitigation)
6. Milestone timeline

Assume a team of [X] people with [describe skills].
Timeline constraint: [X weeks/months].
```

### 9. Weekly Review
```
Help me conduct a weekly review. I'll share what I did this week.
Analyze it and give me:

1. **Wins** — What went well and why
2. **Lessons** — What I learned from failures/challenges
3. **Patterns** — Recurring themes (good or bad)
4. **Next week priorities** — Top 3 things to focus on
5. **One thing to stop doing** — Biggest time/energy waste

This week: [describe your week]
```

### 10. Negotiate Like a Pro
```
I need to negotiate [describe situation — salary, contract, price, etc.].

Give me:
1. My BATNA (Best Alternative to Negotiated Agreement)
2. Opening position and target position
3. Key leverage points I have
4. Likely objections and how to counter each
5. Specific phrases and scripts I can use
6. Walk-away point
7. Non-monetary items I can trade

Context: [provide details about the negotiation]
```

## Learning & Education

### 11. Explain Like I'm 5 / 15 / Expert
```
Explain [complex topic] at three levels:

1. **ELI5 (Age 5):** Use a simple analogy or story a child would understand
2. **ELI15 (High School):** Use proper terms but explain each one
3. **Expert Level:** Assume deep domain knowledge, include nuances and edge cases

For each level, include one concrete example.
```

### 12. Feynman Technique
```
I want to learn [topic]. Use the Feynman Technique:
1. Explain the concept in simple language (no jargon)
2. Identify the gaps — where does the simple explanation break down?
3. Go back and fill those gaps with clearer explanations
4. Create an analogy that captures the essence

Then give me 5 practice questions (easy → hard) to test my understanding.
```

### 13. Study Plan Generator
```
Create a comprehensive study plan for learning [subject/skill].
Time available: [X hours per week] for [X weeks].
Current level: [beginner/intermediate/advanced].
Goal: [what you want to achieve].

Include:
- Weekly topics and subtopics
- Recommended resources (books, courses, videos)
- Practice exercises for each week
- Milestones and self-assessment checkpoints
- Common pitfalls to avoid
```

### 14. Textbook Chapter Summary
```
Summarize the key concepts from this text in a structured format:
1. **Main Ideas** (3-5 bullet points)
2. **Key Terms** (term: definition)
3. **Formulas/Frameworks** (if applicable)
4. **How It Connects** (to related topics)
5. **Exam-style questions** (5 questions with answers)
6. **One-paragraph summary** (for quick review)

Text: [paste content]
```

## Creative & Writing

### 15. Story Builder
```
Help me develop a story with these elements:
- Genre: [genre]
- Setting: [time and place]
- Protagonist: [brief description]
- Central conflict: [the problem]
- Tone: [dark/light/humorous/dramatic]

Create:
1. A compelling opening paragraph (hook the reader)
2. A three-act structure outline
3. Three plot twists I could use
4. Character arc for the protagonist
5. A powerful closing line
```

### 16. Analogy Generator
```
Generate 5 creative analogies to explain [concept] to [audience].
Each analogy should:
- Use a different source domain (sports, cooking, music, nature, etc.)
- Be vivid and memorable
- Capture the most important aspect of the concept
- Be culturally accessible

Rank them from most effective to least effective and explain why.
```

### 17. Blog Post Framework
```
Write a blog post outline for: "[topic]"
Target audience: [who]
Goal: [inform / persuade / entertain / how-to]
Length: [word count]

Provide:
1. 5 alternative titles (SEO-optimized)
2. Hook/opening paragraph
3. Subheadings with key points under each
4. Where to include data/examples/stories
5. Call-to-action at the end
6. Meta description (155 characters)
```

## Research & Analysis

### 18. Research Assistant
```
I'm researching [topic]. Act as a research assistant:

1. What are the key questions I should be investigating?
2. What are the major schools of thought or perspectives?
3. Who are the leading researchers/thinkers?
4. What are the most cited/influential works?
5. What gaps exist in current knowledge?
6. What methodology would you recommend?
7. What are the potential biases to watch for?

Provide a structured research brief I can use as a starting point.
```

### 19. SWOT Analysis
```
Conduct a thorough SWOT analysis for [company/product/strategy]:

**Strengths:** Internal advantages (at least 5)
**Weaknesses:** Internal disadvantages (at least 5)
**Opportunities:** External favorable factors (at least 5)
**Threats:** External unfavorable factors (at least 5)

For each point, be specific (not generic). Include market data where relevant.
End with 3 strategic recommendations based on the analysis.
```

### 20. Competitive Analysis
```
Compare [Product/Company A] vs [Product/Company B] vs [Product/Company C]:

Create a comparison table covering:
- Target audience
- Key features (top 10)
- Pricing
- Strengths
- Weaknesses
- Market positioning
- User reviews sentiment

Then give me: Who should choose which and why?
```

## Meta-Prompts

### 21. Prompt Improver
```
I want to achieve [goal] using AI. Here's my current prompt:
"[your prompt]"

Improve this prompt by:
1. Making it more specific and unambiguous
2. Adding context the AI needs
3. Specifying the output format
4. Adding constraints that improve quality
5. Including examples of good output

Give me the improved prompt ready to copy-paste.
```

### 22. System Prompt Designer
```
Design a system prompt for an AI assistant that:
- Role: [what the AI should be — e.g., "a senior Python developer"]
- Personality: [tone and style — e.g., "concise, no fluff, uses analogies"]
- Constraints: [what it should/shouldn't do]
- Output format: [how it should structure responses]
- Knowledge boundaries: [what to admit it doesn't know]

Make it under 500 words. Include 2-3 example interactions showing
ideal behavior.
```

### 23. Chain of Thought
```
Solve this problem step by step. For each step:
1. State what you're doing and why
2. Show your work
3. Check if the result makes sense before moving on
4. If you're uncertain, say so and explain your reasoning

At the end, verify your answer by approaching the problem differently
or checking edge cases.

Problem: [your problem]
```
