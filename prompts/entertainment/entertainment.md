# Entertainment Prompts

> Prompts for games, stories, roleplay, humor, trivia, puzzles, and creative fun.

**Best with:** ChatGPT, Claude, any creative LLM

---

## Interactive Games

### 1. Text Adventure Game
```
Create a text adventure game set in [setting: haunted mansion / space station /
medieval kingdom / cyberpunk city].

Rules:
- Describe each scene vividly (2-3 sentences)
- Give me 3-4 choices each turn
- Track my inventory and health
- Include puzzles, combat, and NPC dialogue
- Add surprise events and plot twists
- The game should have at least 3 possible endings

Start the game now. Set the scene and give me my first choices.
```

### 2. 20 Questions AI
```
Let's play 20 Questions. I'm thinking of [a person / place / thing / concept].
You ask yes/no questions one at a time. After each answer, tell me your
top 3 guesses and confidence level. Try to guess in as few questions as
possible. Keep count of questions asked.

You go first — ask your first question.
```

### 3. Riddle Master
```
Give me 10 riddles, progressively harder:
- Riddles 1-3: Easy (kids can solve)
- Riddles 4-6: Medium (need to think)
- Riddles 7-9: Hard (tricky lateral thinking)
- Riddle 10: Expert (very challenging)

Format each as:
**Riddle [N]:** [The riddle]
(Wait for my answer before revealing the solution)

Make them original, not the classic ones everyone knows.
```

### 4. Trivia Night Host
```
Host a trivia night with 5 rounds, 5 questions each:

Round 1: [Category — e.g., Science]
Round 2: [Category — e.g., Pop Culture]
Round 3: [Category — e.g., History]
Round 4: [Category — e.g., Geography]
Round 5: Lightning Round (mixed, rapid fire)

For each question:
- Clear question
- Multiple choice (A, B, C, D)
- Difficulty indicator (Easy/Medium/Hard)
- Fun fact after the answer

Keep score. Include a bonus question worth double points.
```

### 5. Would You Rather (AI Edition)
```
Let's play "Would You Rather" with interesting AI/tech themed scenarios.
Give me dilemmas one at a time. After I choose, give me:
1. An interesting perspective on my choice
2. What percentage of people typically choose each option
3. A follow-up consequence of my choice
4. The next dilemma

Make them creative, thought-provoking, and sometimes funny.
Start now!
```

## Storytelling & Fiction

### 6. Collaborative Story
```
Let's write a story together. We'll take turns — you write a paragraph,
then I write a paragraph.

Genre: [choose genre]
Setting: [choose setting]
Tone: [serious / humorous / mysterious / romantic]

Rules:
- Each turn is 3-5 sentences
- End each turn on a hook or question
- Build on what the other person wrote
- Introduce unexpected elements

You start with the opening paragraph.
```

### 7. Character Generator
```
Create 5 unique, compelling characters for a [genre] story:

For each character provide:
- **Name and age**
- **Appearance** (2 sentences, distinctive features)
- **Personality** (3 defining traits, 1 flaw)
- **Backstory** (3 sentences — what shaped them)
- **Secret** (something no one else knows)
- **Speech pattern** (how they talk — formal, slang, quiet, loud)
- **Sample dialogue** (one line that captures their essence)

Make them diverse and create interesting dynamics between them.
```

### 8. Plot Twist Generator
```
I'm writing a story where [describe your current plot].

Generate 5 plot twists, ranging from subtle to mind-blowing:

1. **The Small Twist** — A surprising but believable revelation
2. **The Character Twist** — Someone is not who they seem
3. **The World Twist** — The setting/reality isn't what we thought
4. **The Moral Twist** — The villain was right, or the hero was wrong
5. **The Meta Twist** — Breaks conventions of the genre itself

For each: How to set it up (plant clues early) and the payoff moment.
```

### 9. Alternate History
```
Write a compelling alternate history scenario where [change one historical event]:

Example: "What if the Library of Alexandria was never destroyed?"

Cover:
1. The immediate aftermath (first year)
2. The ripple effects (next decade)
3. How the modern world would be different
4. Which technologies would be ahead/behind
5. Unexpected consequences nobody would predict
6. A day-in-the-life vignette of someone in this timeline

Be creative but historically grounded.
```

## Humor & Fun

### 10. Stand-Up Comedy Bit
```
Write a 3-minute stand-up comedy bit about [topic].

Style: [observational / self-deprecating / absurdist / deadpan]
Audience: [describe audience]

Structure:
- Strong opening joke (grab attention)
- Setup → punchline → tag (callback or extra laugh)
- Build through 3-4 connected jokes
- Callback to an earlier joke
- Strong closer (biggest laugh)

Write it as a script with [pause], [gesture], and [audience reaction] notes.
```

### 11. Roast Generator
```
Write a friendly roast about [topic/person/thing] in the style of
a comedy roast (fun, not mean).

Include:
- 5 one-liner roast jokes
- 1 longer setup-punchline joke
- 1 backhanded compliment
- 1 self-deprecating callback

Keep it fun, clever, and PG-13. No punching down.
```

### 12. Dad Joke Machine
```
Generate 15 original dad jokes on the theme of [topic].

Requirements:
- Must be groan-worthy but clever
- Mix of puns, wordplay, and anti-humor
- Family-friendly
- Original (not the classics everyone's heard)

Rate each joke on the Dad Joke Scale:
- Light chuckle
- Eye roll
- "Get out"
- Standing ovation groan
```

## Puzzles & Brain Teasers

### 13. Logic Puzzle Creator
```
Create an original logic puzzle (Einstein's riddle style):

- 5 categories (e.g., names, colors, pets, drinks, houses)
- 5 items in each category
- 10-15 clues that uniquely determine the solution
- The clues should require deduction (not just direct statements)

Present the puzzle cleanly, then provide:
- Step-by-step solution walkthrough
- Which clues are redundant (if any)
- Difficulty rating
```

### 14. Escape Room Designer
```
Design a virtual escape room with theme: [theme].

Time limit: 60 minutes
Difficulty: [beginner / intermediate / expert]

Create:
- Room description (what do you see when you enter?)
- 5 puzzles in sequence
- Each puzzle: description, clue system, solution
- Hidden items to find
- How puzzles connect to the final solution
- 3 red herrings (false leads)
- Hint system (3 levels per puzzle)

The final puzzle should combine elements from all previous puzzles.
```

### 15. Mystery Scenario
```
Create a murder mystery dinner party scenario:

Setting: [location and time period]
Victim: [brief description]
Number of suspects: 6

For each suspect:
- Name and role
- Motive (why they'd want the victim dead)
- Alibi (with a hole in it)
- Clue they're connected to
- One secret unrelated to the murder (red herring)

The solution should be deducible from the clues if you pay attention.
Include a host script with when to reveal each clue.
```
