# Business Prompts

> Prompts for strategy, operations, finance, startups, and business analysis.

**Best with:** ChatGPT, Claude, any capable LLM

---

## Strategy

### 1. Business Model Canvas
```
Create a Business Model Canvas for [business idea/company]:
- Customer Segments
- Value Propositions
- Channels
- Customer Relationships
- Revenue Streams
- Key Resources
- Key Activities
- Key Partnerships
- Cost Structure

For each block, provide 3-5 specific items (not generic). Then identify
the riskiest assumption and suggest how to validate it quickly.
```

### 2. Market Entry Strategy
```
I want to enter the [market/industry] with [product/service].
Current market size: [if known].
My advantage: [what makes us different].

Analyze:
1. Market attractiveness (Porter's Five Forces)
2. Top 5 competitors and their positioning
3. Entry barriers and how to overcome them
4. Go-to-market strategy (first 90 days)
5. Pricing strategy with justification
6. First 3 customer segments to target and why
7. Key metrics to track in the first 6 months
```

### 3. Startup Pitch Deck Script
```
Write a pitch deck script for my startup:
- Problem: [the problem you solve]
- Solution: [your product/service]
- Target market: [who you serve]
- Traction: [any numbers — users, revenue, growth]
- Ask: [how much funding, what it's for]

Create 10-12 slides with:
- Slide title
- Key message (1 sentence)
- Supporting data/visual description
- Speaker notes (what to say, 30-60 seconds per slide)

Keep it compelling. Investor attention span is short.
```

### 4. OKRs Generator
```
Create quarterly OKRs for [team/company] based on our strategy:
- Company mission: [your mission]
- Current challenges: [describe challenges]
- Growth targets: [your targets]

Generate 3 Objectives, each with 3-4 Key Results.
Each Key Result must be:
- Measurable (specific number)
- Time-bound (this quarter)
- Ambitious but achievable
- Under the team's control

Include leading indicators, not just lagging ones.
```

## Finance

### 5. Financial Model Builder
```
Build a simple financial model for [business type]:
- Revenue model: [how you make money]
- Current monthly revenue: [amount or $0 if pre-revenue]
- Growth assumption: [expected growth rate]
- Key costs: [list main costs]

Create a 12-month projection table showing:
- Revenue (with assumptions)
- COGS
- Gross margin
- Operating expenses (itemized)
- EBITDA
- Burn rate and runway (if startup)

Format as a clear markdown table. Flag the most sensitive assumptions.
```

### 6. Unit Economics Calculator
```
Calculate the unit economics for my [business model]:
- Customer Acquisition Cost inputs: [ad spend, sales team cost, etc.]
- Average Revenue per User: [amount and frequency]
- Churn rate: [monthly/annual]
- Gross margin: [percentage]
- Time to payback: [if known]

Compute: CAC, LTV, LTV:CAC ratio, payback period, contribution margin.
Tell me if the business is viable and what to improve first.
```

## Operations

### 7. Process Optimizer
```
Here's our current process for [describe process]:
[Step 1] → [Step 2] → [Step 3] → ...

Analyze this process:
1. Identify bottlenecks (where does it slow down?)
2. Find redundancies (what's duplicated or unnecessary?)
3. Automation opportunities (what can be automated with current tools?)
4. Error-prone steps (where do mistakes happen?)
5. Redesigned process (show the improved flow)
6. Expected improvement (time saved, error reduction)
```

### 8. Hiring Plan
```
I need to hire for [role]. Create:

1. Job description (responsibilities, qualifications, nice-to-haves)
2. Scorecard with 5 key competencies to evaluate
3. Interview plan (4 rounds with different focus each)
4. Top 10 interview questions with what-good-looks-like answers
5. Take-home assignment or case study prompt
6. Red flags to watch for
7. Compensation benchmarking approach

Company context: [size, stage, industry, location]
```

### 9. Customer Feedback Analyzer
```
Analyze these customer feedback responses and provide:

1. **Sentiment Breakdown** — Positive/Neutral/Negative percentages
2. **Top 5 Themes** — Most frequently mentioned topics
3. **Feature Requests** — Ranked by frequency and urgency
4. **Pain Points** — What's causing frustration
5. **Praise** — What customers love (protect this)
6. **Churn Risk Signals** — Which responses indicate potential churn
7. **Action Items** — Top 3 things to do based on this feedback

Feedback data: [paste feedback]
```

## Marketing & Sales

### 10. Value Proposition Canvas
```
Create a Value Proposition Canvas for [product/service]:

**Customer Profile:**
- Jobs to be done (functional, social, emotional)
- Pains (frustrations, risks, obstacles)
- Gains (desires, benefits, expectations)

**Value Map:**
- Products & services (what we offer)
- Pain relievers (how we address each pain)
- Gain creators (how we create each gain)

Map the fit: which pains and gains do we address best?
Where are the gaps we should fill?
```

### 11. Sales Objection Handler
```
I'm selling [product/service] to [target customer].
Price: [amount].

Give me responses to these common objections:
1. "It's too expensive"
2. "We're happy with our current solution"
3. "We need to think about it"
4. "Can you give us a discount?"
5. "We don't have budget right now"
6. "Your competitor offers [X] that you don't"
7. "I need to check with my [boss/team/partner]"

For each: Acknowledge → Reframe → Respond → Next step.
Keep responses conversational, not scripted.
```

### 12. Content Calendar
```
Create a 4-week content calendar for [brand/company]:
- Platform: [LinkedIn / Twitter / Instagram / Blog / All]
- Industry: [your industry]
- Goals: [brand awareness / leads / engagement / thought leadership]
- Posting frequency: [X times per week]

For each post:
- Date and platform
- Content type (text, image, video, carousel, poll)
- Topic and hook
- Key message
- Call to action
- Hashtags (if applicable)

Include a mix of educational, promotional, and engagement content (80/20 rule).
```
