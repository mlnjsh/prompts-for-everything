# Marketing Prompts

> Prompts for digital marketing, SEO, ad copy, brand strategy, and growth.

**Best with:** ChatGPT, Claude, any LLM

---

## Brand & Strategy

### 1. Brand Voice Guide
```
Create a brand voice guide for [brand/company]:
Industry: [industry]
Target audience: [describe ideal customer]
Brand values: [list 3-5 values]
Competitors: [list main competitors]

Include:
1. Brand personality (3 adjectives that define the voice)
2. Tone spectrum (formal ↔ casual, serious ↔ playful)
3. Do's and Don'ts (specific language to use and avoid)
4. Sample copy in our voice for: social post, email, error message, FAQ answer
5. Words we always use / never use (10 each)
6. How voice changes by channel (social vs email vs support)
```

### 2. Competitor Messaging Analysis
```
Analyze the messaging of these 3 competitors: [list competitors]

For each, identify:
1. Their positioning statement (what they claim to be)
2. Primary value proposition
3. Target audience (based on messaging)
4. Emotional appeals used
5. Key phrases and power words
6. Strengths of their messaging
7. Gaps we can exploit

Then: Write our counter-positioning — how we differentiate from all three.
```

### 3. Customer Persona Builder
```
Create a detailed buyer persona for [product/service]:

Demographics: [age range, income, location, job]
Psychographics: [values, interests, lifestyle]

Include:
1. Name and photo description
2. Day-in-the-life scenario
3. Goals (professional and personal)
4. Frustrations and pain points (specific, not generic)
5. How they discover new products
6. Decision-making process (who influences them)
7. Objections to buying from us
8. Content they consume (platforms, formats, topics)
9. Quotes they'd say about their problem
10. How to reach them (channels, timing, messaging)
```

## SEO & Content

### 4. SEO Content Brief
```
Create an SEO content brief for the keyword: "[target keyword]"

Include:
1. Search intent analysis (informational/transactional/navigational)
2. Suggested title (with keyword, under 60 chars)
3. Meta description (with keyword, under 155 chars)
4. H2 and H3 subheading structure
5. Word count target (based on competing content)
6. Related keywords to include naturally (LSI terms)
7. Internal linking opportunities
8. External reference sources to cite
9. Featured snippet opportunity (format to target)
10. Content angle that beats current top results
```

### 5. Blog Topic Cluster
```
Build a topic cluster around the pillar topic: "[main topic]"

Pillar page: [title and brief description]

Create 10 cluster articles:
- Title (SEO-optimized)
- Target keyword
- Search intent
- Brief outline (3-4 H2s)
- How it links back to the pillar page
- Estimated monthly search volume bracket (high/medium/low)

Organize into a content hierarchy showing relationships between articles.
```

### 6. SEO Audit Checklist
```
Perform an SEO audit checklist for [website/page URL description]:

Technical SEO:
- Page speed issues
- Mobile responsiveness
- Crawlability (robots.txt, sitemap)
- HTTPS, canonical tags, structured data
- Core Web Vitals

On-Page SEO:
- Title tag, meta description optimization
- Heading hierarchy (H1, H2, H3)
- Keyword density and placement
- Image alt tags, file names
- Internal linking structure

Content SEO:
- Content quality and depth
- E-E-A-T signals
- Content freshness
- Featured snippet optimization

Off-Page:
- Backlink profile assessment approach
- Social signals
- Brand mentions

Prioritize fixes by impact and effort.
```

## Ad Copy

### 7. Google Ads Copy
```
Write Google Ads copy for [product/service]:

Target keyword: [keyword]
Landing page offer: [what they get]
Unique selling point: [what makes us different]

Create 5 ad variations:
Each with:
- Headline 1 (30 chars max) — include keyword
- Headline 2 (30 chars max) — benefit or CTA
- Headline 3 (30 chars max) — trust signal or urgency
- Description 1 (90 chars max) — expand on value
- Description 2 (90 chars max) — CTA + differentiator

Variations: feature-focused, benefit-focused, urgency-focused,
social-proof-focused, question-based.
```

### 8. Facebook/Instagram Ad Copy
```
Write 5 Facebook/Instagram ad copy variations for [product]:

Target audience: [describe]
Objective: [awareness / traffic / conversions]
Offer: [discount, free trial, lead magnet]

For each variation:
- Hook (first line — must stop the scroll)
- Body (3-5 lines, benefit-driven)
- CTA (clear action)
- Suggested image/video description

Ad styles:
1. Problem-agitate-solve
2. Social proof / testimonial style
3. Before and after
4. Direct offer with urgency
5. Story-based (mini narrative)

Include emoji suggestions (use sparingly).
```

### 9. Email Subject Lines
```
Generate 20 email subject lines for [campaign type]:

Goal: [open rate optimization]
Audience: [describe]
Email content: [brief description of email]

Create 4 categories of 5 each:
1. Curiosity-driven (open loops, questions)
2. Benefit-driven (what they'll gain)
3. Urgency-driven (time-sensitive, FOMO)
4. Personal/conversational (feels human)

For each: A/B test variant, character count, and predicted relative performance.
```

## Social Media

### 10. Viral Post Formula
```
Create 5 social media posts designed for maximum engagement on [platform]:

Topic: [your topic]
Brand: [your brand]
Goal: [engagement / shares / saves / clicks]

Use proven viral formulas:
1. Contrarian take ("Unpopular opinion: [X]")
2. Thread/carousel educational post
3. "I [did X] for [time]. Here's what happened."
4. Relatable observation + twist
5. Data-driven insight + visual

For each: post text, visual description, best posting time, hashtag strategy.
```

### 11. LinkedIn Content Strategy
```
Create a 4-week LinkedIn content calendar for [professional/brand]:

Industry: [industry]
Goal: [thought leadership / lead generation / personal brand]
Posting frequency: [3-5x per week]

Week 1: Establish expertise (educational content)
Week 2: Show behind the scenes (authenticity)
Week 3: Engage community (polls, questions, debates)
Week 4: Convert (case study, testimonial, soft sell)

For each post:
- Day and time
- Post type (text, carousel, video, poll, article)
- Hook (first 2 lines)
- Full post draft
- CTA
- Engagement prompt (question to ask)
```

## Analytics & Growth

### 12. Growth Experiment Design
```
Design a growth experiment for [metric to improve]:

Current state: [current metric value]
Target: [goal metric value]
Timeline: [experiment duration]

Provide:
1. Hypothesis (If we [action], then [outcome] because [reason])
2. Experiment design (what changes, what stays the same)
3. Success metrics (primary and secondary)
4. Sample size calculation
5. Implementation plan
6. Risks and how to mitigate them
7. Expected learnings regardless of outcome

Follow the ICE framework: Impact (1-10), Confidence (1-10), Ease (1-10).
```
