# Writing Prompts

> Prompts for creative writing, copywriting, technical writing, and content creation.

**Best with:** ChatGPT, Claude, any LLM

---

## Creative Writing

### 1. Story Opener Generator
```
Write 5 compelling opening lines for a [genre] story about [theme].

Each opener should use a different technique:
1. In media res (start in the middle of action)
2. A provocative statement or question
3. Sensory immersion (sight, sound, smell)
4. Dialogue that reveals character and conflict
5. An ominous or mysterious tone-setter

For each, write the opening paragraph (3-4 sentences) that follows naturally.
```

### 2. Character Voice Workshop
```
I have a character who is [describe character: age, background, personality].

Write the same scene — [describe scene] — in this character's voice.
Then rewrite it from a completely different character's perspective:
[describe second character].

Show how voice, word choice, sentence structure, and observations
change based on who is narrating. Point out the specific techniques
you used to differentiate the voices.
```

### 3. Show Don't Tell Converter
```
Convert these "telling" sentences into "showing" prose:

1. "She was angry." → [show through action, dialogue, physical sensation]
2. "The room was messy." → [show through specific details]
3. "He was nervous." → [show through body language and thoughts]
4. "They were in love." → [show through interaction and subtext]
5. "It was a scary night." → [show through atmosphere and sensory detail]

For each, write 2-3 sentences that show instead of tell.
Then explain the technique used.
```

### 4. Dialogue Workshop
```
Write a dialogue scene between [Character A] and [Character B].

Situation: [describe the conflict or topic]
Subtext: [what they really mean but aren't saying]

Rules:
- No dialogue tags beyond "said" (show emotion through action beats)
- Each character should have a distinct speech pattern
- Include what characters are DOING while talking (body language)
- Build tension — it should escalate
- End on a line that changes everything

Write 15-20 lines of dialogue.
```

## Copywriting

### 5. Landing Page Copy
```
Write landing page copy for [product/service]:

Target audience: [who]
Main benefit: [the #1 reason to buy]
Price: [amount]
Competitors: [who are they losing to]

Include:
1. **Headline** — Benefit-driven, under 10 words
2. **Subheadline** — Clarify the headline
3. **Hero section** — 2-3 sentences of hook copy
4. **3 Feature blocks** — Feature → Benefit → Proof for each
5. **Social proof section** — Testimonial prompts
6. **FAQ** — Top 5 objections as questions, answered
7. **CTA** — Primary button text + supporting text
8. **Urgency element** — Why act now

Tone: [professional / casual / bold / friendly]
```

### 6. Email Sequence
```
Write a 5-email welcome sequence for [product/service]:

Email 1 (Day 0): Welcome — deliver the promised lead magnet, set expectations
Email 2 (Day 2): Value — teach something useful, build credibility
Email 3 (Day 4): Story — share origin story or customer success
Email 4 (Day 7): Soft sell — introduce the product as a solution
Email 5 (Day 10): Hard sell — direct CTA with urgency/scarcity

For each email:
- Subject line (+ 2 A/B test alternatives)
- Preview text
- Body copy (200-300 words)
- CTA button text

Conversion goal: [sign up / purchase / book a call]
```

### 7. Product Description
```
Write 3 versions of a product description for [product]:

Version 1: **Feature-focused** (specifications, technical details)
Version 2: **Benefit-focused** (how it improves the customer's life)
Version 3: **Story-focused** (narrative of someone using the product)

Each version: 100-150 words.
Include: sensory language, power words, and a clear CTA.
Avoid: clichés, vague claims, "best in class" type language.
```

### 8. Social Media Captions
```
Write 10 social media captions for [brand/topic]:

Platform: [Instagram / LinkedIn / Twitter/X]
Tone: [professional / witty / inspirational / educational]

For each:
- Caption text (platform-appropriate length)
- Hook (first line must stop the scroll)
- Call to action
- 5 relevant hashtags
- Best time to post (general)

Mix: 3 educational, 3 engagement, 2 promotional, 2 personal/behind-the-scenes.
```

## Technical Writing

### 9. API Documentation
```
Write developer documentation for this API endpoint:

Endpoint: [method] [path]
Purpose: [what it does]
Auth: [how authentication works]

Include:
1. Description (2-3 sentences, when to use this endpoint)
2. Request (method, URL, headers, body with all parameters)
3. Response (success + error responses with status codes)
4. Code examples in [Python, JavaScript, cURL]
5. Rate limits and pagination
6. Common errors and troubleshooting

Style: Clear, scannable, developer-friendly. Use tables for parameters.
```

### 10. README Template
```
Write a README.md for my project: [project name]

What it does: [brief description]
Tech stack: [languages, frameworks]
Target users: [who would use this]

Include:
1. Project title with badges (build status, version, license)
2. One-paragraph description
3. Screenshot/demo GIF placeholder
4. Quick start (install in 3 steps or less)
5. Usage examples (3 common use cases with code)
6. Configuration options (table format)
7. Contributing guidelines (brief)
8. License

Keep it concise. README should take <2 minutes to read.
```

### 11. Technical Blog Post
```
Write a technical blog post: "[title]"

Audience: [junior / mid / senior developers]
Length: [1000 / 1500 / 2000 words]

Structure:
1. Hook — Why should they care? (problem they face)
2. Context — Brief background (assume smart but unfamiliar)
3. Solution — Step-by-step with code snippets
4. Deep dive — Explain the "why" behind the approach
5. Gotchas — Common mistakes and how to avoid them
6. Conclusion — Summary + next steps + further reading

Include: Working code examples, diagrams (described), and benchmarks if relevant.
No filler paragraphs. Every sentence should teach something.
```

## Content Strategy

### 12. Content Pillar Framework
```
Design a content pillar strategy for [brand/niche]:

Create 4 content pillars with:
- Pillar name and description
- Why it matters to the audience
- 10 content ideas per pillar (mix of formats)
- Which platform each idea works best on
- How pillars interconnect

Content types to include across pillars:
- How-to guides, listicles, opinion pieces
- Case studies, interviews, data-driven
- Quick tips, long-form deep dives, video scripts

Total: 40 content ideas organized by pillar and priority.
```

### 13. Newsletter Framework
```
Design a weekly newsletter for [topic/industry]:

Name: [suggest 5 options]
Sections (keep to 4-5):
1. [Section name] — What goes here, target length
2. [Section name] — What goes here, target length
3. [Section name] — What goes here, target length
4. [Section name] — What goes here, target length

For the first edition, write the complete content.
Include: subject line, preview text, intro paragraph,
each section filled out, sign-off, and CTA.

Target reading time: 5 minutes.
```
