# Music & Audio Prompts

> Prompts for AI music generation, voice cloning, sound design, and audio production.

**Best with:** Suno, Udio, ElevenLabs, Stable Audio, AIVA

---

## Song Generation (Suno / Udio)

### 1. Pop Song
```
Style: Upbeat pop, 120 BPM, major key
Mood: Feel-good summer anthem
Instruments: Synth bass, acoustic guitar, claps, bright synths
Vocals: Female vocalist, catchy melody, layered harmonies in chorus
Structure: Verse - Pre-chorus - Chorus - Verse - Pre-chorus - Chorus - Bridge - Final Chorus
Lyrics theme: [your theme — road trip / falling in love / new beginnings]
```

### 2. Lo-Fi Hip Hop
```
Style: Lo-fi hip hop, chill beats to study to
BPM: 85, swung rhythm
Elements: Vinyl crackle, jazzy piano chords, mellow bass, tape hiss,
soft hi-hats, rain ambience in the background
Mood: Nostalgic, cozy, late-night studying
No vocals, instrumental only
Duration: 3 minutes with subtle variations
```

### 3. Epic Cinematic Score
```
Style: Epic orchestral cinematic score, Hans Zimmer inspired
Mood: Building from quiet tension to triumphant climax
Instruments: Full orchestra — strings section, French horns, timpani,
choir, bass drums, brass crescendo
Structure: Quiet piano intro → strings enter → brass builds →
full orchestra climax → quiet resolution
Use case: Movie trailer or video game boss battle
```

### 4. Indian Classical Fusion
```
Style: Indian classical fusion with electronic elements
Raga: Yaman (evening raga, peaceful mood)
Instruments: Sitar melody, tabla rhythm, electronic bass,
ambient pads, tanpura drone
BPM: 90-100, build gradually
Mood: Meditative then energetic
Blend traditional Indian melodies with modern production
```

### 5. Jazz Standard
```
Style: Jazz, smooth late-night club atmosphere
Instruments: Piano trio — upright bass walking lines, brushed drums,
piano comping with occasional solo runs
Key: Bb major, with jazz chord extensions (7ths, 9ths, 13ths)
Mood: Sophisticated, romantic, smoky jazz club
BPM: 120, medium swing feel
Include a piano solo section with bebop-influenced runs
```

### 6. Rock Anthem
```
Style: Arena rock anthem, stadium-filling energy
Instruments: Distorted electric guitars (rhythm and lead), punchy bass,
driving drums, organ, crowd clap in chorus
BPM: 130, straight beat
Structure: Guitar riff intro → Verse (clean) → Chorus (full distortion)
→ Verse → Chorus → Guitar solo → Final chorus with key change
Mood: Powerful, rebellious, sing-along
```

### 7. Ambient / Meditation
```
Style: Ambient meditation music, deep relaxation
Elements: Soft drones, singing bowls, gentle wind sounds, distant
wind chimes, slow evolving pads
Key: C major or A minor, no dissonance
BPM: None (free-flowing, no strict tempo)
Mood: Peaceful, transcendent, timeless
Duration: 10 minutes, very gradual evolution
No sudden changes or sharp sounds
```

## Voice & Speech (ElevenLabs)

### 8. Podcast Intro
```
Voice style: Warm, authoritative male voice, NPR quality
Pacing: Medium-slow, confident
Emotion: Welcoming, professional

Text: "Welcome to [Podcast Name]. I'm [Host Name], and each week
we explore [topic] with the world's leading experts. In today's
episode, we're diving into [specific topic]. Let's get started."
```

### 9. Audiobook Narrator
```
Voice style: Rich baritone, classic audiobook narrator
Pacing: Deliberate, with natural pauses for emphasis
Emotion: Shifts with the text — serious for exposition,
animated for dialogue, tense for suspense

Text: [paste your passage]

Character voices: Give slight vocal changes for different characters
(deeper for older characters, lighter for younger ones).
```

### 10. E-Learning Narration
```
Voice style: Clear, friendly female voice, educational tone
Pacing: Moderate, slightly slower than conversational
Emphasis: Stress key terms and important concepts
Tone: Encouraging, patient, like a great teacher

Text: "In this module, we'll learn about [topic]. The key concept
to understand is [concept]. Let me walk you through this step by step.
First, [step 1]. Notice how [observation]. This is important because
[reason]."
```

## Sound Design

### 11. Game Sound Effects
```
Create descriptions for sound effects for a [genre] video game:

1. Player footsteps (on: stone / grass / metal / water)
2. Weapon sounds (sword swing, arrow release, magic spell)
3. UI sounds (menu click, item pickup, quest complete)
4. Ambient loops (forest, cave, marketplace, dungeon)
5. Enemy sounds (growl, attack, death)
6. Music stingers (victory, defeat, discovery, danger)

For each: describe the character, duration, frequency range, and feel.
```

### 12. Podcast Soundscape
```
Design the audio production for a [genre] podcast:

1. Intro music (10 seconds) — describe mood and instruments
2. Transition sounds between segments — describe 3 variations
3. Background ambience for different segments
4. Outro music (15 seconds) — describe the closing feel
5. Stinger for important moments
6. Subscribe reminder jingle

Keep the sonic identity consistent. All elements should feel like
they belong to the same brand.
```

## Prompt Modifiers for Music AI

### Genre Keywords
| Genre | Keywords to Add |
|-------|----------------|
| Pop | `catchy hooks, polished production, radio-friendly` |
| Rock | `distorted guitars, driving drums, raw energy` |
| Jazz | `swing rhythm, walking bass, chord extensions, improvisation` |
| Classical | `orchestral, dynamics, counterpoint, sonata form` |
| Electronic | `synthesizers, 808 bass, sidechain compression, drops` |
| Hip Hop | `boom bap, 808s, sampling, hard-hitting kicks` |
| R&B | `smooth vocals, neo-soul chords, groove, falsetto` |
| Country | `acoustic guitar, pedal steel, storytelling, twang` |
| Folk | `acoustic, fingerpicking, harmony vocals, storytelling` |
| Metal | `blast beats, palm muting, drop tuning, growls` |

### Mood Keywords
- **Uplifting:** `bright, hopeful, euphoric, triumphant`
- **Sad:** `melancholic, bittersweet, longing, somber`
- **Energetic:** `driving, intense, powerful, explosive`
- **Chill:** `relaxed, dreamy, floating, mellow`
- **Dark:** `brooding, ominous, haunting, sinister`
- **Romantic:** `intimate, warm, tender, passionate`

### Production Quality
- `"Studio quality, professionally mixed and mastered"`
- `"Lo-fi, warm tape saturation, vinyl crackle"`
- `"Live recording feel, room ambience, natural reverb"`
- `"Overproduced, maximalist, wall of sound"`
- `"Minimal, sparse arrangement, space between notes"`
