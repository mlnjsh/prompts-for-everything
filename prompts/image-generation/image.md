## Prompt Formula & Modifiers

### The Master Formula
```
[Subject] + [Action/Pose] + [Setting/Background] + [Lighting] + [Style/Medium]
+ [Camera/Lens] + [Mood/Atmosphere] + [Quality modifiers] + [Aspect ratio]
```

### Style Keywords
| Style | Keywords |
|-------|---------|
| Photorealistic | `photorealistic, 8K, hyperdetailed, RAW photo, DSLR` |
| Oil Painting | `oil painting, impasto, classical, museum quality` |
| Watercolor | `watercolor, soft edges, paper texture, wet on wet` |
| Digital Art | `digital painting, concept art, artstation, trending` |
| Anime | `anime style, cel shading, Studio Ghibli, vibrant` |
| 3D Render | `octane render, unreal engine, ray tracing, cinema 4D` |
| Pencil Sketch | `graphite pencil, detailed sketch, cross-hatching` |
| Pixel Art | `pixel art, 16-bit, retro game sprite, limited palette` |
| Pop Art | `pop art, Andy Warhol style, bold colors, halftone dots` |
| Art Nouveau | `art nouveau, Alphonse Mucha, ornamental, flowing lines` |

### Lighting Keywords
| Effect | Keywords |
|--------|---------|
| Dramatic | `chiaroscuro, Rembrandt lighting, dramatic shadows` |
| Soft | `diffused light, overcast, soft box, gentle shadows` |
| Backlit | `backlit, rim light, silhouette, halo effect` |
| Neon | `neon lighting, glow, cyberpunk, vibrant night` |
| Natural | `golden hour, natural light, sun-dappled, outdoor` |
| Studio | `three-point lighting, key light, fill light, softbox` |

### Negative Prompts (Stable Diffusion)
```
ugly, blurry, low quality, deformed, disfigured, extra limbs, bad anatomy,
bad hands, missing fingers, cropped, worst quality, low resolution, watermark,
signature, text, logo, out of frame, duplicate
```

### Midjourney Parameters
- `--ar 16:9` — Widescreen landscape
- `--ar 9:16` — Portrait/mobile
- `--ar 1:1` — Square
- `--ar 2:3` — Vertical portrait
- `--v 6.1` — Version 6.1 (latest)
- `--s 250` — Stylize (higher = more artistic)
- `--c 20` — Chaos (higher = more varied)
- `--q 2` — Quality (higher = more detail)
- `--no [item]` — Exclude something from the image
